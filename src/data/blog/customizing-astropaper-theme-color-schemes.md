This post will explain how you can enable/disable light & dark mode for the website.

## Table of contents

## Enable/disable light & dark mode

AstroPaper theme will include light and dark mode by default. This default behavior can be disabled in `SITE` configuration object.

## Customize color schemes

Both light & dark color schemes of AstroPaper theme can be customized in the `global.css` file.
